# Airbnb API

REST & GraphQL API of the Airbnb Clone using Django REST Framework and Graphene GraphQL

### API Actions

- [x] JWT authentication
- [x] List Rooms
- [x] Login
- [x] Create Account
- [x] See Room
- [x] Add Room to Favourites
- [X] See Profile
- [X] Edit Profile
- [x] Create room
- [x] Delete Room
- [ ] Search Rooms
